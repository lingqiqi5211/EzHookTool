MethodQuery

class MethodQuery

方法查询条件。

用在 findMethod、findMethodOrNull、findAllMethods 的查询块里。 多个条件会同时生效，全部满足才算匹配。

val method = clazz.findMethod {
    name("foo")
    paramCount(2)
    returnType(String::class.java)
}
