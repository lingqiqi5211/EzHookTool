FieldQuery

class FieldQuery

字段查询条件。

用在 findField、findFieldOrNull、findAllFields 的查询块里。 多个条件会同时生效，全部满足才算匹配。

val field = clazz.findField {
    name("mContext")
    type(Context::class.java)
}
