ConstructorQuery

class ConstructorQuery

构造器查询条件。

用在 findConstructor、findConstructorOrNull、findAllConstructors 的查询块里。 多个条件会同时生效，全部满足才算匹配。

val constructor = clazz.findConstructor {
    paramCount(2)
    params(String::class.java, Int::class.java)
}
