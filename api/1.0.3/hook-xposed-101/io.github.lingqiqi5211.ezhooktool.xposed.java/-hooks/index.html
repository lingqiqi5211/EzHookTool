Hooks

object Hooks

供 Java 调用的 hook 入口。

已经拿到 Method / Constructor 时使用 createHook。 想按名称和参数类型查找并立即 hook 时使用 findAndHookMethod 或 findAndHookConstructor。

findAndHookMethod 和 findAndHookConstructor 的最后一个可变参数必须是 IMethodHook、IReplaceHook 或 libxposed 的 Hooker。
